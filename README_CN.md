### 简介
这是一个功能强大的《无主之地4》存档修改器，旨在提供全面的存档编辑功能。完全免费，支持中英文切换。
该工具允许你修改角色数据、背包物品、解锁游戏内容，即时生成和定制武器装备，并提供强大的代码转换和生成功能。

---

### 主要功能

#### 存档管理
- 自动扫描游戏存档目录。
- 支持 Steam 和 Epic 平台的 64位 ID 解密与加密。
- 支持 `.sav` 文件的打开、保存及另存为。
- 自动备份原存档。

#### 角色编辑
- 修改角色名称、当前职业。
- 自定义难度设置。
- 修改角色等级、经验值 (XP)。
- 修改专精等级 (Spec Level) 和专精点数 (Spec Points)。
- 修改金钱 (Money) 和镒矿 (Eridium)。
- 一键同步背包内所有物品等级至当前角色等级。

#### 解锁与预设
提供多种一键解锁功能：
- 移除地图迷雾、发现所有地点、解锁所有安全屋。
- 解锁所有收集品、完成所有挑战、完成所有成就。
- 跳过剧情任务、跳过所有任务。
- 一键满级 (50级)、最大化储物格代币 (SDU)。
- 解锁密藏大门、解锁所有载具、解锁所有专精、解锁 UVHM 模式。
- 解锁/最大化所有内容 (Unlock/Max Everything)。

#### 物品与背包管理
- 查看和管理背包中的物品。
- 为物品设置 Flag（普通、收藏、垃圾、编组1-4）。
- 直接添加 Base85 或解码格式的物品代码到背包。
- 从背包中读取物品进行修改或复制序列号。

#### 装备生成与编辑
拥有独立的标签页，用于深度定制各类装备：
- **武器编辑器 (Weapon Editor)**: 修改现有武器的部件、皮肤、元素、受福等。
- **武器生成器 (Weapon Gen)**: 从头生成自定义武器。
- **职业模组 (Class Mod)**: 自定义职业、稀有度、技能加成、被动专长。
- **强化模组 (Enhancement)**: 自定义厂商、稀有度、专长堆叠。
- **手雷 (Grenade)**: 自定义厂商、等级、稀有度、专长、元素、引信等。
- **护盾 (Shield)**: 自定义厂商、等级、稀有度、护盾类型（能量/护甲）、专长。
- **修复套件 (Repkit)**: 自定义前缀、抗性、固件、专长。
- **重武器 (Heavy Weapon)**: 自定义枪管、元素、固件、附件等。
- 所有生成器均支持直接“加入背包”或“复制序列号”。

#### 转换器与高级工具
- **代码转换**: 支持 Base85 编码与 Deserialize (解析后) 数据互转。
- **批量处理**: 批量转换代码，批量将代码列表导入背包。
- **遍历生成器**: 支持通过设定数值范围批量生成物品序列（如生成所有皮肤、遍历所有部件组合）。
- **YAML 编辑器**: 提供树状视图和文本视图，直接编辑存档的原始 YAML 数据结构，适合高级用户。

### 如何构建 (Build)

如果你想从源码构建可执行文件 (.exe)，请确保已安装 Python，并运行项目根目录下的 `pyinstaller_config.py` 脚本。

该脚本会自动处理依赖、收集所需的资源文件（如图片、JSON、CSV 等），并调用 PyInstaller 生成 `dist/BL4SaveEditor.exe`。

**步骤:**

1.  安装依赖:
    ```bash
    pip install pyinstaller pillow pyyaml pycryptodome PyQt6 pandas
    ```
2.  运行构建脚本:
    ```bash
    python pyinstaller_config.py
    ```
3.  脚本运行完成后，生成的可执行文件将位于 `dist` 文件夹中。

### 使用说明
1. 打开软件，程序会自动扫描默认存档路径。
2. 选择存档并根据提示输入对应的 64位 ID（通常会自动检测，如需手动输入请填入 Steam ID 或 Epic ID）。
3. 加载成功后，即可在各个标签页进行修改。
4. 修改完成后点击“保存”即可生效。
5. 建议在修改前手动备份存档文件，虽然软件会自动创建备份。
